# Production Evidence Checklist

Use this file as the public proof trail for recruiter and interview review. Keep claims tied to artifacts generated by the pipeline; do not replace synthetic README rows until the logs below come from real historical or live data.

## 1. Walk-Forward Backtest

Goal: show that forecasts and allocation decisions were evaluated only on future slates.

Required artifacts:

- `betting_system/data/processed/backtest_log.jsonl`
- `docs/calibration_plot.png`
- README backtest table updated from the log, with date range and data source named

Runbook:

```bash
export PYTHONPATH="$(pwd)"
dk-pipeline --dry-run
python scripts/plot_calibration.py
```

For production numbers, run the backtest against real historical features and model artifacts through `betting_system.pipeline.backtest.walk_forward_backtest`. Archive the JSONL log before editing the README table.

Minimum fields to report:

- Date range and market type
- ROI on staked capital
- Total staked and total profit
- Max drawdown
- Brier score or calibration plot generated from holdout predictions
- Baseline comparison, if `backtest.baseline_compare` is enabled

## 2. Live Market Feed

Goal: show the system can refresh an edge-ranked feed without notebook steps.

Required artifacts:

- `betting_system/data/processed/market_opportunities.json`
- `betting_system/data/processed/picks_today.json`
- Screenshot or Streamlit URL showing the feed status badge

Runbook:

```bash
export PYTHONPATH="$(pwd)"
python -m betting_system.pipeline.run_market_pipeline
dk-pipeline
```

If live APIs are unavailable, the dashboard will use fixture fallback. Keep the `is_live` metadata visible so demo and production runs are not mixed.

## 3. Production Readiness Notes

Add these only when measured:

- API latency percentiles for `/picks/today` and `/markets/opportunities`
- Daily refresh schedule and retrain cadence
- Model artifact version and config hash for each published run
- Known limits, including API quota, sport coverage, and market liquidity filters
